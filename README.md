# Flip Mobile Automation - WebdriverIO

Project ini merupakan automation testing untuk aplikasi Flip menggunakan **WebdriverIO**, **Appium**, dan **Android Emulator**.

## Tools yang Digunakan
- WebdriverIO
- Appium
- Node.js
- Android Emulator

## Test Scenario
Automation yang dibuat mencakup:

- Login menggunakan nomor telepon
- Input PIN pada halaman login
- Verifikasi bahwa proses login dapat dijalankan
